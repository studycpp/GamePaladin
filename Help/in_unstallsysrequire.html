系统需求

  • 基于Pentium II, Pentium III, Pentium IV, 或同规格的 AMD 处理器的个人计算机

  • Microsoft Windows 98 (初版和第二版), Windows Millennium Edition (ME),或 Windows 2000 或Windows XP或更高

  • 最小32 MB 内存, 建议64 MB RAM 以上

  • 临时文件会在低阶搜索 时产生.额外的硬盘空间依据任务变化. 推荐剩余 64 MB 以上.最好256MB以上。

安装

江湖行GamePaladin 是"绿色软件".双击可执行自解压文件或用Winrar3.0及其以上版本右键安装'GPSetup.exe'.

卸载

删除江湖行GamePaladin所在的目录及其所有子目录、文件
