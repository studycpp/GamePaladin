江湖行GamePaladin 帮助_存档文件低阶搜索向导

www.AdvNetsoft.com

存档文件低阶搜索 很类似 内存低阶搜索.

1. 打开启动游戏.

2. 加入和选择存档文件搜索/编辑任务;

3.输入'?'(默认) 或 ' + ' 或 ' - ' 或 ' ! ' 或 ' = '  在输入文本框，回车 或 点击 搜索 按钮.

这一步骤仅仅初始化搜索过程.( 备份存档文件 ).

4.继续玩游戏. 在游戏里, 试着增加、减少、改变，不变数据 .对于文档的普通（高阶）、低阶搜索 ,你必须进行游戏存档 .

5.按键 Alt + Tab 或 主界面弹出热键 切换到江湖行 GamePaladin. 根据数值变化输入 ' + ' 或 ' - ' 或 ' ! ' 或 ' = ' 在文本框，回车或者点击搜索 按钮搜索.

相对增加
+
相对减少
-
相对变化
!
相对不变
=

6.点击返回游戏 或 按键Alt + Tab 切换回游戏；重复步骤4~5 直到找到数据在文件中的地址.

7.在文件搜索结果列表中 双击或点击右键，加入文件地址锁定列表.

8.切换回游戏，载入存档。你会发现数值修改了.
