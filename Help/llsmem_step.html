江湖行GamePaladin 帮助_内存低阶搜索向导

www.AdvNetsoft.com

1. 运行游戏.

2.为该游戏 加入和选择内存搜索任务 ;

3.输入 '?'(默认) 或 ' + ' 或 ' - ' 或 ' ! ' 或 ' = ' 在文本框中；回车 或者点击搜索 按钮开始搜索.

这一步骤初始化搜索过程 .(捕获当前游戏进程内存全部数据)

4.继续玩游戏. 在游戏里，试着增加，减少，变化，不变(不推荐，因为不变的数据太多了，除非已找到的结果少)欲修改的数值 .

5.按键 Alt + Tab 或者 主界面弹出热键 切换回江湖行GamePaladin. 根据数值变化在输入文本框输入 ' + ' 或 ' - ' 或 ' ! ' 或 ' = ' ；；回车 或者点击搜索 按钮搜索.

相对增加
+
相对减少
-
相对变化
!
相对不变
=

6.点击返回游戏 或按键Alt + Tab 切换回游戏.重复步骤4~5 直到找到数据在内存中的地址.

7.点击鼠标右键或者双击搜索结果列表项目 。选择修改的模式，或者锁定.

注意: 你可以存储和载入锁定地址列表记录(*.gpx).
